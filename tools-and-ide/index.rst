Tools and IDE
=============

The `SmartThings IDE <http://graph.api.smartthings.com>`__ (Integrated Development Environment) provides SmartThings developers with a set of tools to manage their SmartThings account, and build and publish custom SmartApps and Device Type Handlers.

In this guide, you will learn:

- The various tools that allow you to manage your Locations, Hubs, Groups, and Devices.
- How to use the IDE and simulator to build and test SmartApps and Device Type Handlers.
